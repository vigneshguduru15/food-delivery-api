# FastAPI Food Delivery App

## Project Overview
This is a FastAPI backend application for a food delivery system.

## Features
- Menu APIs
- Cart system
- Order workflow
- Search, sort, pagination
- Combined browse API

## Tech Stack
- FastAPI
- Python
- Pydantic

## Run Project
uvicorn main:app --reload
